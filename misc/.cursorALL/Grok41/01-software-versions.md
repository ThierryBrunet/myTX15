# Software Versions (as of February 2026)

When assisting with RadioMaster TX15 configuration, always reference these latest stable (non-RC) versions:

- **EdgeTX firmware**: 2.11.4 ("Jolly Mon") – download the color-screen build for RadioMaster radios from https://github.com/EdgeTX/edgetx/releases
- **ExpressLRS firmware**: 4.0.0 – target appropriate hardware (2.4GHz) via the Configurator
- **ExpressLRS Configurator**: 1.7.11 – download from https://github.com/ExpressLRS/ExpressLRS-Configurator/releases

If newer stable versions exist, prompt the user before using them.
